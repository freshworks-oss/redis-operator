# Maintainers

This repository is maintained by contributors with merge rights on
[github.com/freshworks/redis-operator](https://github.com/freshworks/redis-operator).
The authoritative roster of people with write access is managed in GitHub
repository settings and may change over time.

## Roles

- **Maintainers** can approve and merge pull requests, cut releases, and manage
  repository settings as permitted by the owning organization.
- **Contributors** submit pull requests, reviews, and issues; anyone in the
  community can contribute.

## Maintainer expectations

- Review pull requests in a timely manner when possible (best effort within
  about a week is a reasonable target).
- Ask for tests or documentation when a change warrants it.
- Follow the [Security policy](SECURITY.md) when handling reports.

## Becoming a maintainer

There is no formal application process. Contributors who consistently provide
high-quality changes, thoughtful review, and constructive collaboration may be
invited by existing maintainers as the project needs grow.
